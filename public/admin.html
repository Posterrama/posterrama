Display Settings Help

Transition Interval

How long each poster is displayed before switching to the next one (1–300 seconds). Shorter intervals are more dynamic; longer intervals allow more time to read details.

Tip: A range of 8–15s usually feels smooth without being too fast.

Note: Wallart Mode uses its own refresh logic and ignores the transition interval.

Wallart Mode

  • Enable Wallart Mode: Switch to a multi‑poster grid layout.
  • Poster Density: Controls how many posters are visible at once.
  • Poster Refresh Rate: How frequently grid tiles update.
  • Timing Randomness: Adds variation to refresh timing for a natural look.
  • Animation Type: Choose a specific animation or use random.

Tip: A randomness of 2–4 and refresh rate around 3–6 gives a balanced, natural flow.

Cinema Mode

Portrait‑style digital movie poster mode. You can set the display orientation to Auto or a fixed Portrait option.

Visual Elements

  • Toggle ClearLogo, Poster, and Metadata visibility.
  • Show Rotten Tomatoes badge and set a minimum score threshold.

Clock Widget

Shows a clock in screensaver mode. Choose the timezone (Auto or specific) and 24h/12h format.

UI Element Scaling

Resize content, clearlogo, clock, and global UI scale. Use presets for quick device‑specific setups or Reset to defaults.

Presets: Try 4K TV for living‑room distances, and Ultrawide for 21:9 monitors; then fine‑tune Global to 110–120% if needed.

Effects & Transitions

Select the transition effect (e.g., Ken Burns, Fade, Slide) and adjust the pause time between effects.

Media Sources & Filters

  • Connect Plex or Jellyfin and choose which libraries to include (Movies/Shows).
  • Use genre filters and rating thresholds to fine‑tune what appears.
  • Recently Added: show only new content from the last N days.

Tip: Start with one source first, verify posters load, then enable others.

Devices (Basics)

  • Status: Offline (grey), Online (green), Live (blue)
  • Controls: Previous/Next, Play/Pause, Pin poster, Reload
  • Overrides: Apply a preset or JSON override per device
  • Sync: Devices align on a global tick for smooth changes

Device management is desktop‑only; on mobile you’ll see a short notice.
